# Revivo - Site Vitrine (Phase 1)

Site vitrine développé en Node.js + Express + EJS pour la plateforme Revivo.

## ▶️ Lancer le site en local

### Prérequis
- Node.js installé
- npm

### Commandes

```bash
npm install
npm start
```

Puis ouvrir : http://localhost:3000
